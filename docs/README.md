## 🎯 このドキュメントの目的

わんこめをカスタマイズしたい方のために、以下の情報を提供します。

- **プラグイン開発**: Node.js を使用したプラグインの作成方法
- **CSS カスタマイズ**: OBS 用のコメント表示テンプレートのカスタマイズ方法
- **API リファレンス**: わんこめの公開 API（OneSDK、プラグイン API）の仕様
- **設定ガイド**: わんこめアプリの基本的な使い方とカスタマイズ方法

## 📌 クイックスタート

### プラグインを作りたい

1. [プラグイン開発ガイド](./plugin-development.md) を読む
2. サンプルプラグイン（waiting-count）を参考にする
3. わんこめのプラグインディレクトリに配置して動作確認

### CSS テンプレートをカスタマイズしたい

1. [CSS テンプレートカスタマイズガイド](./css-customization.md) を読む
2. サンプルテンプレート（simple-css、neon-custome）を参考にする
3. わんこめアプリでカスタムテンプレートとして読み込み

### API 仕様を知りたい

1. [わんこめ API リファレンス](./onecomme-api.md) を読む
2. OneSDK の使い方を確認
3. プラグイン API のライフサイクルを理解
